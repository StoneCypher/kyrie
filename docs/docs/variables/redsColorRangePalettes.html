    Reds Color Range Palette Collection

    This file contains color palettes that combine reds with other colors across the spectrum. Each palette uses reds more heavily (60-70%) than the secondary color (30-40%). Both light and dark variants are provided for different background contexts.

    Light variants use darker colors suitable for light backgrounds. Dark variants use lighter colors suitable for dark backgrounds.
